# Domain docs

This repository uses a single domain context.

Before exploring or modifying the model, analysis, or manuscript:

- read `CONTEXT.md` at the repository root if it exists;
- read relevant decisions under `docs/adr/` if they exist;
- use the vocabulary established by those documents;
- surface any conflict with an existing decision instead of silently overriding it.

If the context or ADR files do not exist, proceed without requiring them. They should be created only when domain terminology or a durable scientific/modeling decision needs to be recorded.
